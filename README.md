🎮 Game Arena - The Ultimate Gaming Experience

Game Arena is a modern, high-performance landing page designed specifically for the gaming and eSports community. Built with a focus on immersive UX and sleek "Dark Mode" aesthetics, this project showcases advanced CSS techniques and responsive layouts.

🚀 Key Features

Fully Responsive Design: Optimized for all devices, from mobile phones to large desktop monitors.

Smooth Navigation: Integrated Smooth Scroll for a seamless transition between sections.

Interactive Game Slider: A custom-styled Bootstrap Carousel featuring hover zoom effects and dynamic content overlays.

Infinite Sponsors Scroll: A continuous, high-performance CSS animation for partner logos.

Professional Team Section: Interactive member cards with grayscale-to-color transitions and hover effects.

Optimized Contact Form: A clean, user-friendly form with custom-styled inputs and focus states.

🛠️ Tech Stack

HTML5: Semantic structure for better SEO and accessibility.

CSS3: Advanced styling including Flexbox, Grid, Linear Gradients, and Keyframe Animations.

Bootstrap 5: For a robust responsive grid system and pre-built components.

JavaScript (Vanilla): To handle carousel controls, auto-closing the mobile navbar, and smooth scrolling logic.

Font Awesome 7: High-quality icons for an authentic gaming look.

Google Fonts: Utilizing Chakra Petch for body text and Days One for headers.
